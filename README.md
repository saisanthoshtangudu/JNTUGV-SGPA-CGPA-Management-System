# JNTUGV SGPA & CGPA Management System

A Python-based academic record management system that calculates SGPA and CGPA, stores student details in text files, and provides search and view functionalities through a menu-driven interface.

## Features

* Add Student Records
* Calculate Semester SGPA
* Calculate Overall CGPA
* Save Records Using File Handling
* Search Student by Hall Ticket Number
* View Individual or All Student Records
* Exception Handling for Missing Files
* Menu-Driven Program

## Technologies Used

* Python
* File Handling
* Functions
* Lists
* Dictionaries
* Exception Handling

## How to Run

```bash
python main.py
```
